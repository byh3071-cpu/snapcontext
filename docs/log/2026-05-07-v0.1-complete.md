# 작업 로그 — SnapContext v0.1 완료

- **캡쳐:** Visible / Element / Document, Side Panel 미리보기, `CAPTURE_RESULT`에 URL·title·viewport·UA·이미지 크기 메타 포함.
- **핀:** 추가·메모·삭제·재번호, 미리보기 오버레이.
