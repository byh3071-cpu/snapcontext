# SnapContext 개인정보처리방침

최종 수정일: 2026-05-24

공개 URL (Chrome Web Store / Whale Store 제출용):

https://github.com/byh3071-cpu/SnapContext/blob/master/docs/PRIVACY.md

## 수집하는 정보

SnapContext는 개인정보를 수집하지 않습니다.

## 데이터 저장

- 캡처 이미지, 핀 주석, 설정값은 사용자의 브라우저 로컬 저장소(`chrome.storage.local`)에만 저장됩니다.
- 외부 서버로 전송되는 데이터는 없습니다.

## 권한 사용 목적

| 권한 | 목적 |
|---|---|
| sidePanel | 캡처 도구 UI를 사이드 패널에 표시하기 위해 사용 |
| storage | 설정값과 캡처 히스토리를 로컬에 저장하기 위해 사용 |
| scripting | 페이지에 콘텐츠 스크립트를 주입하여 요소 선택·본문 영역 탐색을 수행하기 위해 사용 |
| downloads | PNG 파일을 저장하기 위해 사용 |
| tabs | 현재 탭의 URL·제목 등 메타데이터를 읽기 위해 사용 |
| windows | 캡처 대상 창을 식별하기 위해 사용 |
| host_permissions (`<all_urls>`) | 모든 웹 페이지에서 화면 캡처 및 콘텐츠 스크립트 실행을 허용하기 위해 사용 |

## 제3자 공유

사용자 데이터를 제3자와 공유하지 않습니다.
