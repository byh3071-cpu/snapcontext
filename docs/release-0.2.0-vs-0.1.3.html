새로 생긴 것
  • 01 익명 공유 링크 캡처 이미지와 메모를 링크 하나로. 받은 사람은 로그인·앱 설치 없이 브라우저에서 바로 봅니다.
  • 02 7일 뒤 자동으로 안 열림 공유 링크는 만든 지 7일이 지나면 막힙니다. 영구 공개가 아니라 한시적 공유라 안심.
  • 03 실수로 민감정보가 새지 않게 부가정보(주소·메모) 포함은 기본 꺼짐이고, 처음 공유할 때 동의를 한 번 받습니다. 로그인 토큰·쿠키 같은 건 절대 함께 전송되지 않습니다.
  • 04 받는 사람 화면이 안전·깔끔 공유 페이지는 이미지와 설명을 보기 좋게 보여주고, 악성 코드가 끼어들 수 없게 막아놨습니다.
  • 05 앱 화면 정돈 사이드패널·버튼 디자인을 더 깔끔하고 일관되게 다듬었습니다.
안심 포인트
  • ✓새로 요구하는 권한 없음. 업데이트해도 “이 권한을 추가로 허용할까요?” 같은 게 안 뜹니다. 0.1.3과 동일한 권한으로 공유 기능만 더해진 것.
  • ✓만든 링크가 외부(다른 기기·시크릿창)에서 실제로 열리는 것까지 확인 완료.
