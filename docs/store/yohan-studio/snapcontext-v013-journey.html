기술 결정 메모

  • host_permissions <all_urls> — 사이드패널 UX에서 activeTab 한계 (ADR 002). CWS 심사 지연 가능.
  • 원격 코드 없음 — MV3 번들만 패키지에 포함.
  • 데이터 로컬 only — chrome.storage.local, 외부 전송 없음.
  • Whale 단축키 — suggested_key 미적용 시 shortcuts에 V/E/M/G 수동 등록.
