### 2️⃣ 답글 — UI/UX 리디자인  *(첨부: `new-panel.png`)*

```text
가장 크게 바뀐 건 사이드패널입니다.

예전: 보라↔네이비 그라데이션, "AI가 만든 듯한" 느낌

포커스 링·모션 접근성(prefers-reduced-motion)까지 챙겼습니다.
```

### 3️⃣ 답글 — 익명 공유 (신규 핵심 기능)  *(첨부: `feature-share.png`)*

```text
이번 버전의 메인 신규 기능: 로그인 없는 익명 공유 🔗

캡처를 7일짜리 익명 링크로 바로 공유할 수 있어요.
· 계정·로그인·추적 없음
· 7일 뒤 접근 차단 + 서버에서도 자동 영구 삭제
· '컨텍스트 포함'은 기본 꺼짐 + 최초 1회 동의제
· 토큰·쿠키·로컬 데이터는 전송 안 함 (누출 차단 E2E 검증)

디자인 피드백을 화면째 던질 때 편합니다.
```

### 4️⃣ 답글 — 어떻게 쓰나  *(첨부: `feature-context.png`)*

```text
쓰는 법은 단순합니다.

1. 캡처 — 영역 / 요소 / 문서 본문 / 풀페이지 스크롤 전체
2. 캡처 위에 핀 찍고 메모 → 주석까지 프롬프트에 자동 포함
3. ChatGPT·Claude에 붙여넣거나, 익명 링크로 공유

버그 리포트·리팩토링·레퍼런스 템플릿도 들어 있어요.
설명을 다시 쓸 필요가 없습니다.
```
